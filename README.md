# RadioVision-3D
RadioVision 3D est un prototype d'application web destiné aux patients en radiothérapie, conçu pour leur permettre de visualiser en 3D l'évolution de leur tumeur au fil des séances de traitement, de manière intuitive et accessible — sans nécessiter l'interprétation médicale complexe d'imageries traditionnelles (scanner, IRM, radiographies).
